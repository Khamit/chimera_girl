# Chimera Girl - Spirit Hunters 🦊👻

**2D action-platformer** на C++17 + Raylib для macOS (Apple Silicon).

Три школьницы-охотницы против духов в мире японского фольклора.
Собирай энергию, трансформируйся в зверей, сражайся с волнами врагов.

---

## 🎮 Управление

| Клавиша | Действие |
|---------|----------|
| WASD / Стрелки | Движение |
| Space / W / ↑ | Прыжок |
| J / ЛКМ | Атака |
| 1–4 | Трансформация |
| E | Меню улучшений |

## 🐺 Формы

- **1** — Человек (0 энергии)
- **2** — Кролик (20) — скорость, двойной прыжок
- **3** — Волк (50) — мощная атака
- **4** — Змея (100) — огромный урон и HP

## ⚡ Энергия

Собирай предметы на уровне:
- 🪙 Монеты — +8 энергии
- 💎 Алмазы — +18 энергии
- ⭐ Звёзды — +28 энергии
- ❤️ Жизни — восстанавливают HP
